# Rick and Morty

Simple Android app fully in Kotlin with Clean Architecture able to list characters from the world of
Rick And Morty.

Jonathan Drini
https://www.linkedin.com/in/jonathan-drini/

## Features

- Characters list
- Character details

## Libraries

- Jetpack Compose
- Coroutines
- Hilt
- Coil
- Navigation
- Retrofit
- OkHttp3
- Moshi
